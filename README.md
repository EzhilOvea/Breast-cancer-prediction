Breast Cancer Diagnostic Classification 🏥This project uses Logistic Regression to predict whether a breast mass is Malignant (cancerous) or Benign (healthy) based on clinical measurements of cell nuclei.📌 Project OverviewEarly diagnosis of breast cancer can significantly improve prognosis and survival rates. This project demonstrates a machine learning workflow to classify tumors with high accuracy using a classic medical dataset.Key Learning Objectives:Data preprocessing and cleaning using Pandas.Managing class imbalance with Stratified Splitting.Understanding the Sigmoid Function in Binary Classification.Handling feature dimensions with NumPy Reshaping.📊 Dataset InformationThe dataset used is the Breast Cancer Wisconsin (Diagnostic) Dataset. It includes 30 numerical features such as:Radius (mean of distances from center to points on the perimeter)Texture (standard deviation of gray-scale values)Perimeter & AreaSmoothness (local variation in radius lengths)Target Labels:0: Malignant1: Benign🛠️ Technologies UsedLanguage: Python 3.xLibraries: * pandas (Data manipulation)numpy (Numerical operations & array reshaping)sklearn (Machine Learning modeling)🚀 How to RunClone this repository:Bashgit clone https://github.com/[Your-Username]/breast-cancer-prediction.git
